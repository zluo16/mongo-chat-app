MongoChat

Very basic chat app built with MongoDB and socket.io

### To run program

Install dependencies:

'''bash
git clone git@github.com:zluo16/mongo-chat-app.git
cd backend
npm install
cd ..
mongo-chat
yarn install
cd..
'''

Run both servers

In backend run:

'''bash
npm start
'''

In mongo-chat run:

'''bash
yarn start
'''
